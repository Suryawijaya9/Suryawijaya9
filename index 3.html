CMS

CMS adalah CMS (Content Manajemen System) adalah sebuah perangkat lunak atau sistem yang mengatur konten pada situs web, bisa situs web yang berupa company profile, e-commerce, blog, forum, dan lainnya. Jika kalian mengenal mie instan pada makanan, CMS menjadi pembuatan situs instan yang disediakan di internet. CMS dapat kamu gunakan meskipun kamu tidak mengetahui mengenai pemrograman sama sekali. CMS juga menyediakan banyak template.

JENIS CMS

1. CMS untuk personal web dan personal blog Contoh: joomla, mambo, wordpress Kegunaan untuk membuat website pribadi

2. CMS untuk Online Shop Contoh: prestashop, magento, opencart dll Kegunaan untuk membuat website yang menjual produk dan jasa

3. CMS untuk membuat forum Contoh: phpBB, vBulletin, dll Kegunaan untuk membuat website forum.

KELEBIHAN CMS Menggunakan CMS bisa diumpakan sebagai kita sudah disiapkan rangka website. Website tersebut sudah siap pakai dan tinggal diisi dengan konten dan edit tampilan. Dengan seperti itu tentunya pembuatan website menjadi lebih mudah dan cepat dibanding jika kita membangunnya secara manual. Selain itu, CMS juga memiliki dan menyediakan banyak pilihan template dan support. Terutama jika kamu menggunakan CMS WordPress, yang mana bisa digunakan secara gratis karena sifatnya open source. Banyak tersedia template website disana mulai dari tampilan yang sederhana seperti blog hingga template yang memiliki banyak fitur seperti website toko online dan website berita.

KEKURANGAN CMS

- Tema dan Plugin premium cenderung mahal untuk kebanyakan user dan bisnis

- Semakin banyaknya add-ons & plugin yang diinstal, maka semakin lambat pula loading time website nya.

- Kemiripan website

- Masalah keamanan
